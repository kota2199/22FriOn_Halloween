# あやかしゲーム
こちらは”あやかしゲーム”のUnityプロジェクトです。

ゲームを実行するときはAssets/Scenes/InGameシーンから実行をお願いします。

# Unityroomにて配信中
[こちら](https://unityroom.com/games/ayakashi_game)からブラウザ上でプレイできます。

# 操作方法
## プレイヤーの移動
右移動：Dキー, 画面左下の右ボタン

左移動：Aキー, 画面左下の左ボタン

ピースを落とす：Enter(Return)キー, 画面右下のボタンをクリック

# 使用アセット、フリー素材
